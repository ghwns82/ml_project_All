# 분기별 분석하기

<1. 분석요청 단계>

명령어 : python analyze_long_term.py {검색한 상권명} {검색한 서비스명}

안내사항(규열씨가 알아야 할 것들)
- analyze_long_term.py은 절대경로, 상대경로 모두 인식가능하게 코딩함.

- 분석하고자 하는 상권명과 서비스 명을 입력하면 된다. 상권명과 서비스명을 정확히 입력할 필요가 없으며, 키워드만 입력해도 검색이 가능하다. 
ex) '서울 강남구 강남역'을 '강남역'이라고 쳐도 된다.
ex) '애완동물' 대신 '애완'만 검색해도 된다.

- 만약 검색한 단어를 기준으로 데이터가 없거나 충분하지 않은 경우 에러 메시지를 출력한다.
ex) 데이터가 없거나 부족합니다.

- 분석에 성공하면 다음과 같은 메시지를 출력한다.
ex) 분석에 성공하였습니다.

- 만약 검색할 단어를 모르겠다면, data 폴더에 서울시우리마을가게상권분석서비스_상권명.txt와 서울시우리마을가게상권분석서비스_서비스명.txt 파일에 작성된 상권들과 서비스들을 참고하면 된다. 

특이사항
- 분석으로 산출되는 파일들은 analyze_long_term_work/work_{검색한 상권명}_{검색한 서비스명}/ 에 저장된다.

- 산출물들은 나중에 분석결과를 작성할 때 쓰인다.



<2. 분석한결과 받아보기>

명령어 : python explain_long_term.py {검색한 상권명} {검색한 서비스명}

안내사항(규열씨가 알아야 할 것들)
-  explain_long_term.py은 절대경로, 상대경로 모두 인식가능하게 코딩함.

- 추세를 분석하여 내린 총평을 출력함.
ex) 매출액과 매출건수는 감소하고 있으나, 점포수는 증가하는 추세입니다. 현재 해당 산업은 과잉상태입니다.

- 출력한 총평은 다음과 같은 경로에 저장됨
ex) explain_long_term_work/{검색한 상권명}_{검색한 서비스명}/explain.txt

- 현재는 총평만 출력됨. 만약 그래프나, 다른 분석멘트 등 다양한 결과를 원한다면 추가해야함.
